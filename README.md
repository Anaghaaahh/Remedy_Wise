# Home Remedies Collection

A responsive website for storing and organizing traditional home remedies, beauty tips, wellness advice, and home hacks.

## Features

- 📱 **Fully Responsive Design** - Works beautifully on desktop, tablet, and mobile devices
- 🏷️ **Category Organization** - Organize remedies into three categories:
  - 💄 Beauty Remedies
  - 🌿 Wellness Remedies
  - 🏡 Home Hacks
- 💾 **Local Storage** - All remedies are saved in your browser's local storage
- ➕ **Add/Edit/Delete** - Full CRUD functionality for managing remedies
- 🔍 **Filter by Category** - Easily filter remedies by category
- 📝 **Detailed Information** - Each remedy includes:
  - Title
  - Category
  - Ingredients list
  - Step-by-step instructions
  - Benefits (optional)

## Getting Started

Simply open `index.html` in your web browser. No installation or server required!

The website will automatically load with 10 sample remedies across all categories to get you started.

## Usage

1. **View All Remedies**: Click "All Remedies" to see everything
2. **Filter by Category**: Click on Beauty, Wellness, or Home Hacks to filter
3. **Add New Remedy**: Click the "+ Add Remedy" button
4. **Edit Remedy**: Click the ✏️ icon on any remedy card
5. **Delete Remedy**: Click the 🗑️ icon on any remedy card

## Technical Details

- Pure HTML, CSS, and JavaScript (no dependencies)
- Uses localStorage for data persistence
- Mobile-first responsive design
- Modern, clean UI with smooth animations

## Browser Compatibility

Works on all modern browsers that support:
- HTML5
- CSS3 (Flexbox, Grid)
- ES6 JavaScript
- localStorage API
