# RemedyWise

RemedyWise is a web application that provides traditional home remedies for common health and wellness concerns. It organizes remedies into categories such as beauty care, wellness, pregnancy care, and household hacks through a simple and user-friendly interface.

## Features

- Browse remedies by category
- View detailed remedy information
- Submit feedback
- Responsive user interface
- Backend API for managing remedy data

## Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB
